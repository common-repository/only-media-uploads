== Description ==
A simple plugin to restricts uploads to specified file types only (images: jpg, gif, png | videos: wmv, avi, mpeg, mp4, mkv). 
It does not add any options page.  it does not effect on your site's load speed/ time.
 Its just a tiny plugin which works silently in the background.

== Installation ==
This section describes how to install the plugin and get it working.

1. Unzip and Upload the only-media-uploads folder to your plugin directory.
2. Activate it. and everything will be working.

== Frequently Asked Questions ==
What should i do after activating the plugin?
Nothing! You just need to install and active the plugin. After activating the plugin it will automatically be working.
